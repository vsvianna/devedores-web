# Sistema de Controle de Devedores

Projeto desenvolvido para gerenciamento de clientes e controle de pagamentos.

## Funcionalidades

- Cadastro de usuários
- Login e logout
- Cadastro de clientes
- Registro de compras
- Controle de pagamentos
- Dashboard com indicadores
- Gráficos de vendas
- Relatórios mensais
- Relatórios anuais
- Exportação PDF

## Tecnologias

- Python
- Flask
- PostgreSQL
- Bootstrap
- Chart.js

## Deploy

Render + Neon
